## 读取数据源

在数据源中定义好请求外部API

如果请求不到可以用延时调用来解决，可能是他请求的太快了比输出的方法还快，所以做一个延迟就可以解决了

``` js
const fromdata = this.dataSourceMap['数据源名称']._data.data
```

## 请求

需要在数据源中先定义好请求外部API

然后再调用，传参

``` js
  this.dataSourceMap.数据源名称.load({
    "formDataJson": JSON.stringify(参数)
  }).then(ele => {
    console.log("新增成功")
  })
```
