## 多环境配置

pro：生产环境

dev：开发环境

test：测试环境

---：分隔不同环境

```yaml
# 应用服务 WEB 访问端口
spring:
  profiles:
    active: test # 选择环境
---
spring:
  profiles: pro # 生产环境
server:
  port: 8080
---
spring:
  profiles: dev # 开发环境
server:
  port: 8081
---
spring:
  profiles: test # 测试环境
server:
  port: 8082
```
